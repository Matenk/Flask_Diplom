# Task Management

## Описание проекта

Приложение для управления задачами, которое позволяет пользователям создавать, редактировать и удалять задачи, а также назначать их различным пользователям. Это простое и интуитивно понятное веб-приложение, разработанное с использованием Flask, которое поможет вам оставаться организованным и эффективно управлять своими задачами.

## Основные компоненты

- **Frontend:** HTML/CSS для пользовательского интерфейса.
- **Backend:** Flask приложение для обработки запросов и взаимодействия с базой данных.
- **Database:** SQLite alchemy для хранения данных о задачах и пользователях.

## Требования

- Python 3.x
- Flask
- SQLitealchemy
- pip (для установки зависимостей)

## Установка и запуск

Установите необходимые зависимости:
bash
pip install -r requirements.txt

Запустите приложение:
bash
python main.py

Откройте браузер и перейдите по адресу: http://127.0.0.1:5000/.

Задачи
GET /tasks: Получение списка всех задач .
POST /tasks: Создание новой задачи .
DELETE /tasks/<id>: Удаление задачи (требуется аутентификация).
